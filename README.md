
# Project Title

an android application with the name GOOD TIME
application that displays movie recommendations,
search for movies and movie details.





## Features

- Light/dark mode by user system settings
- Rekommendation Movie
- Search Movie
- Movie Details
- play trailer


## Tech Stack

**MVVM**

**HILT Dagger**

**Retrofit**

**Rxandroid**

**Piccasso**
